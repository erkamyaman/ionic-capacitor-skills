# Ionic Capacitor Skills for Claude

This repository contains skills for AI agents working with Ionic Capacitor mobile development.

## Skills Overview

### ionic-capacitor
Use when users want to create a production-ready Ionic Capacitor mobile app with **Angular, React, or Vue**. Covers:
- Project scaffolding (Angular standalone components, React functional components, Vue Composition API)
- Onboarding flow with fullscreen video background
- RevenueCat paywall with weekly/yearly subscriptions
- AdMob banner ads with premium user detection
- i18n localization (Turkish/English) with framework-appropriate library
- Theme switching (Light/Dark/System) via CSS variables
- Push notifications with @capacitor/push-notifications
- Tab navigation with ion-tabs
- Route guards for onboarding flow
- Framework-appropriate state management (Angular services, React hooks, Vue composables)
- @capacitor/preferences for persistent storage
